Första lektionen

Jag var inte med på föreläsningen idag och har därför inte mycket att säga om innehållet ni gick igenom mer än att det i slidesen står om t.ex. hur man bygger upp ett html-dokument, den går också igenom några viktiga/vanliga taggar/element.

Det finns en hel del olika taggar i html, t.ex. kan man använda sig av "strong" för att göra text tjock. Man kan också använda sig av "b" för samma effekt. Utöver dessa två finns också "em" och "i" för att göra text italic samt "u" för att understryka ett ord.

Vill man skriva text utan att webbläsaren formaterar om den eller om man vill skriva ett stycke kod i webbläsaren och behålla sin indentering kan man använda sig av pre-taggen. Vill man skriva ett stycke kod kan man istället använda sig av code-taggen. 

Detta är text som använder sig utav pre-taggen, då kan man t.ex lägga in            såhär många 
                    mellanslag utan att webbläsaren gör om det till ett enda. Nedan är ett stycke kod i en code-tag.
function countToThree() { for(var i = 0; i < 3; i++) { console.log(i); } }

Ett anchor är en tag som kan redirecta användaren till en länk, t.ex. kan du klicka HÄR för att komma till google i en ny tab eller HÄR för att ladda google i denna tab.

En ul eller unordered list är en lista där varje li (list item) är oordnat, alltså att den inte har en nummerordning. Te.x kan det vara en inköpslista.

  • Ägg
  • Mjölk
  • Bröd
  • Gurka

En ol eller ordered list är en lista där varje li (list item) har en ordning, alltså att varje item har har ett nummer (1, 2, 3 etc).

  1. Objekt
  2. Objekt
  3. Objekt
  4. Objekt
